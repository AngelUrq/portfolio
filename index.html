  • Project

    SpectraUPB - Spectrometry app for smartphones

    The low-cost smartphone spectrometer software is a powerful tool designed to transform a smartphone into a fully functional spectrometer. With this software, you can easily analyze the composition of various substances and materials, including liquids, gases, and solids, with a high degree of accuracy and precision.

    The software is designed to be user-friendly and intuitive, allowing anyone to use it, regardless of their level of expertise in spectroscopy. It features a simple and easy-to-use interface, which enables you to take measurements quickly and easily.

    Check more about the app at the download page.

  • Project

    Automatic Calibration - Recurrent Neural Network

    Using recurrent neural networks (RNNs) we automate the detection of reference lines required for spectrometer calibration, which is a time-consuming process in many scientific fields. The RNNs were trained on a dataset of spectral images collected using cameras of different resolutions, and were able to predict spectral lines with high precision. The method was validated using a 3D-printed spectrometer and compared to the NIST atomic spectra database. The automatic selection of reference lines helps speed up the calibration process, making it suitable for spectrometry applications where speed is critical and calibration needs to be performed frequently.

    Zenteno A., Orellana A., Villazon A., and Ormachea O. Automatic Selection of Reference Lines for Spectrometer Calibration with Recurrent Neural Networks. SmartTech-IC 2022

  • Project

    Spectra Element Detection - Neon and Mercury

    We present an approach that uses a convolutional neural network (CNN) for automatically detecting which element corresponds to a spectral image, such as mercury or neon. CNNs are a type of neural network that is well-suited for image recognition tasks.
    We collected a dataset of spectral images with known element labels to train the neural network. The CNN was able to learn to recognize the unique features of each element's spectral signature and accurately identify them in new images. We tested the model on a set of spectral images with unknown element labels, and it was able to correctly identify the corresponding element with high accuracy.

    Zenteno A., Orellana A., Villazon A., and Ormachea O. Automatic Selection of Reference Lines for Spectrometer Calibration with Recurrent Neural Networks. SmartTech-IC 2022

  • Project

    Space Legacy - NASA Space Apps Challenge

    Space Legacy is a platform that uses an artificial intelligence system that automatically summarizes more than 300,000 scientific documents published by NASA, using a visualization feature with embeddings that helps to identify similar articles based on their content. This platform is powered by Large Language Models to enhance the summarization and search capabilities of the system. This made space information more accessible to researchers and scientists, enabling them to easily find related scientific articles.

    For more information, check out the demo.

  • Project

    Drone diffusion model for Super‑Resolution

    Implemented a diffusion model from scratch for super-resolution on drone images as part of a research project. Key contributions include implementing a U-Net architecture following "Image Super-Resolution via Iterative Refinement" paper, developing training pipelines, diffusion schedules, and evaluating model performance using metrics such as PSNR and SSIM. The project showed that diffusion models were able to perform super-resolution on drone imaging.

  • Project

    Alpaca Tutor: A LLM-Based Chatbot for Assisting Students in STEM Courses

    Developed an LLM-powered chatbot using Unsloth library and Llama 3 8B model as a digital tutor for EPFL students. Performed LoRA Supervised fine-tuning (SFT), and generated a dataset of preference pairs using synthetic data augmentation with chain of thought prompting and OpenAI GPT-4o model. Utilized reinforcement learning from human feedback (RLHF) with direct preference optimization (DPO) for training. Achieved an accuracy of 47\% in answering multiple choice questions from EPFL courses, outperforming a baseline accuracy of 37\%.
