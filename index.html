Featured Project Master's Thesis Feb - Sep 2025

Diffusion-Based Reconstruction Methods for Magnetic Resonance Imaging

MRI scans can be accelerated by undersampling k-space data, but this creates aliased images. My thesis explored using diffusion models as learned priors to recover high-fidelity reconstructions from undersampled measurements.

Background

What is a Diffusion Model?

A diffusion model learns to generate images by reversing a noising process. Starting from pure Gaussian noise, it iteratively denoises to produce realistic samples. Here, a model trained on fully-sampled brain MRIs generates new anatomically plausible images. We use this learned prior to guide reconstruction.

  • Fine-tuning LLMs for Argument Detection

    LoRA fine-tuning of Llama-2, Mistral, and Phi for argument classification in news articles.

    View Details

    Part of the CommPass project at EPFL's Distributed Information Systems Laboratory, this work focuses on the argument detection task: the first step in a polarization-identification pipeline for news articles. Using the Low-Rank Adaptation (LoRA) technique, we fine-tuned three open-source LLMs (LLaMa-2-7B, Mistral-7B, and Phi-1.5) to classify sentences as arguments or non-arguments using the litgpt framework.

    The best-performing model, Mistral-7B, achieved an accuracy of 84.24% and F1-Score of 82.23% on the Stab argument detection dataset. The fine-tuned models were applied to analyze English news articles about real-world media events, extracting argumentative structures to help identify polarization.

  • Automatic Calibration - Recurrent Neural Network

    Automating spectrometer calibration using recurrent neural networks.

    View Details

    Using recurrent neural networks (RNNs) we automate the detection of reference lines required for spectrometer calibration, which is a time-consuming process in many scientific fields. The RNNs were trained on a dataset of spectral images collected using cameras of different resolutions, and were able to predict spectral lines with high precision. The method was validated using a 3D-printed spectrometer and compared to the NIST atomic spectra database. The automatic selection of reference lines helps speed up the calibration process, making it suitable for spectrometry applications where speed is critical and calibration needs to be performed frequently.

    Zenteno A., Orellana A., Villazon A., and Ormachea O. Automatic Selection of Reference Lines for Spectrometer Calibration with Recurrent Neural Networks. SmartTech-IC 2022

  • Spectra Element Detection - Neon and Mercury

    CNN-based automatic element detection from spectral images.

    View Details

    We present an approach that uses a convolutional neural network (CNN) for automatically detecting which element corresponds to a spectral image, such as mercury or neon. CNNs are a type of neural network that is well-suited for image recognition tasks.
    We collected a dataset of spectral images with known element labels to train the neural network. The CNN was able to learn to recognize the unique features of each element's spectral signature and accurately identify them in new images. We tested the model on a set of spectral images with unknown element labels, and it was able to correctly identify the corresponding element with high accuracy.

    Zenteno A., Orellana A., Villazon A., and Ormachea O. Automatic Selection of Reference Lines for Spectrometer Calibration with Recurrent Neural Networks. SmartTech-IC 2022
